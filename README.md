Eseguendo il notebook per il processamento dei dati grezzi, veranno calcolati i file JSON collegati ai file js per la visualizzazione del calendario e dei replay delle singole gare. 
Inoltre, la seconda parte del notebook permette di raccogliere automaticamente le immagini dei piloti a differenza dei circuiti che sono stati raccolti manualmente.
Dopo aver eseguito la trasformazione dei dati grezzi, aprendo l'index.html da Visual Studio Code ed utilizzando lo strumento Open with Live Server, è possibile avviare il sistema di visualizzazione.
